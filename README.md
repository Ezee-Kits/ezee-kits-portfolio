# Ezee Kits Portfolio Website

A modern, responsive, and professionally designed portfolio website built with Django, HTML, CSS, JavaScript, and Bootstrap. This portfolio serves as a central platform for showcasing my technical skills, engineering background, professional experience, certifications, and a diverse collection of software and engineering projects.

The website was developed to provide recruiters, employers, collaborators, and potential clients with a comprehensive overview of my capabilities in Electrical Engineering, Software Development, Data Analysis, Machine Learning, Automation, and Renewable Energy Systems.
